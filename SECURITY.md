# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly. Do not open a public issue.

Contact [@marioprieta on GitHub](https://github.com/marioprieta) with:

- A description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

## Scope

react-native-theme-transition is a client-side UI theming library. It does not handle authentication, network requests, or sensitive data storage.

Relevant security concerns include:

- Dependency vulnerabilities (supply chain)
- Injection via rendered content
- Unsafe dynamic code execution

## Disclosure Policy

We follow coordinated disclosure: confirm the issue, prepare and release a fix, then publicly disclose with credit to the reporter.
